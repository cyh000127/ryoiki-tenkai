# 주술회전 (呪術廻戦) - 술식 & 영역전개 정리

**주의**: 스포일러 포함. 최종화 기준 주요 생득술식과 영역전개를 정리했습니다.  
게임 제작 (STT + 손동작 스킬 대전)용으로 일본어 원문(한자 + 후리가나) 위주로 작성했습니다.

## 1. 주요 생득 술식 (生得術式 / Cursed Technique)

| 사용자              | 술식명 (일본어)                  | 한국어 번역          | 설명 및 주요 기술 |
|---------------------|----------------------------------|----------------------|-------------------|
| **고죠 사토루**    | 無下限呪術 (むかげんじゅじゅつ) | 무하한주술          | 공간에 무한을 구현. **蒼 (あお)**, **赫 (あか)**, **紫 (むらさき)** |
| **후시구로 메구미** | 十種影法術 (とくさのかげぼうじゅつ) | 십종영법술         | 그림자 매개 10종 식신 소환 (玉犬, 鵺, 蝦蟇, 大蛇, 満象, 脱兎 등) |
| **료멘 스쿠나**    | 伏魔御厨子 (ふくまみづし)      | 복마어주자          | 절단 공격 (**解**, **捌**). 신의 불꽃 (**神の火**) |
| **마히토**         | 無為転変 (むいてんぺん)        | 무위전변            | 손으로 영혼 형태 변화 (즉사급) |
| **게토 스구루**    | 呪霊操術 (じゅれいそうじゅつ)  | 주령조술            | 주령 포획·조종. **極ノ番「うずまき」** |
| **노바라 쿠기사키**| 芻霊呪法 (すうれいじゅほう)    | 추령주법            | 짚인형 + 못 + 망치 (**共鳴り**, **簪**) |
| **치소 / 카모 노리토시** | 赤血操術 (せっけつそうじゅつ) | 적혈조술           | 혈액 자유 조작 |
| **젠인 마이 / 요로즈** | 構築術式 (こうちくじゅつしき) | 구축술식            | 물질 창조 |
| **이누마키 토게**  | 呪言 (じゅごん)                 | 주언                | 말로 저주 부여 ("爆ぜろ", "止まれ" 등) |
| **토도 아오이**    | 不義遊戯 (ぶぎゆうぎ)           | 불의유희            | 박수로 위치 교환 (**Boogie Woogie**) |
| **하카리 킨지**    | 座殺博徒 (ざさつばくと)        | 좌살박도            | 도박 룰 기반 무한 주력·재생 |
| **나나미 켄토**    | 十劃呪法 (とおかくじゅほう)     | 십획주법            | 7:3 비율 약점 타격. **瓦落瓦落 (がらがら)** |
| **옷코츠 유타**    | 模倣 (もほう) + リカ             | 모방 + 리카         | 타인 술식 복제 |
| **우라우메**       | 氷凝呪法 (ひょうぎょうじゅほう) | 빙응주법            | 얼음·냉기 조작 |
| **젠인 나오야**    | 投射呪法 (とうしゃじゅほう)     | 투사주법            | 1초를 24프레임으로 나누는 초고속 이동 |

**기타 중요 용어**:
- **反転術式 (はんてんじゅつしき)**: 반전술식 (긍정 에너지 생성, 회복)
- **極ノ番 (ごくのばん)**: 극번 (술식의 최대 출력 기술)
- **天与呪縛 (てんよじゅばく)**: 천여주박 (타고난 신체 능력 극대화)

## 2. 영역전개 (領域展開 / Domain Expansion)

| 사용자              | 영역전개명 (일본어)                  | 한국어 번역             | 효과 요약 |
|---------------------|--------------------------------------|-------------------------|----------|
| **료멘 스쿠나**    | 伏魔御厨子 (ふくまみづし)          | 복마어주자 (Malevolent Shrine) | 열린 영역. 범위 내 무차별 절단 (最強 공격형) |
| **고죠 사토루**    | 無量空処 (むりょうくうしょ)        | 무량공처 (Unlimited Void) | 무한 정보 과부하 → 마비·정지 (제어 최강) |
| **마히토**         | 自閉円頓裹 (じへいえんどんか)      | 자폐원돈과             | 영혼 즉시 변형 |
| **죠고**           | 蓋棺鉄囲山 (がいかんてっちせん)    | 개관철위산             | 초고온 용암·화산 지대 |
| **다곤**           | 蕩蘊平線 (たううんへいせん)        | 탕온평선               | 바다 환경 + 식신 대량 소환 |
| **후시구로 메구미**| 嵌合暗翳庭 (かんごうあんえいてい)  | 감합암예정 (Chimera Shadow Garden) | 그림자 바다 + 식신 무제한 소환 (불완전) |
| **하카리 킨지**    | 座殺博徒 (ざさつばくと)            | 좌살박도 (Idle Death Gamble) | 파친코 도박 룰 기반 무한 재생 |
| **히구루마 히로미**| 誅伏賜死 (ちゅうぶくしし)          | 주복사사 (Deadly Sentencing) | 재판·판결 시스템 |
| **젠인 나오야**    | 時胞月宮殿 (じほうげっきゅうでん)  | 시포월궁전             | 시간 가속·지연 |
| **켄자쿠 (게토)**  | 胎蔵遍野 (たいぞうへんや)          | 태장편야               | 중력 + 바이러스 공격 |
| **옷코츠 유타**    | 真贋相愛 (しんがんそうあい)        | 진가상애               | 리카와 연계된 강력 영역 |

**영역전개 공통 특징**:
- 대부분 **「領域展開！」** 라고 외치며 발동
- **必中 (ひっちゅう)** 효과 (영역 안에서는 술식이 반드시 명중)
- 손인(掌印)은 불교·신도 인장을 기반으로 함 (고죠: 제석천인, 메구미: 약사여래인 등)

## 게임 제작 팁 (STT + 손동작)

- **STT 트리거 추천**:  
  - `"無下限呪術"` 또는 `"むかげんじゅじゅつ・蒼!"`  
  - `"領域展開！伏魔御厨子！"`

- **손동작 키 포인트**:  
  - 토도 아오이 → 박수  
  - 고죠 무하한 → 특정 손가락 모양  
  - 영역전개 → 각 캐릭터별 독특한 인(印)
