# v4 선행조건

## 제품 조건

- 일본어 시동 명령은 앱 진입 편의 기능이며 전투 스킬 발동이 아니다.
- 특정 작품 고유 표현은 기본값에서 제외한다.
- 명령어는 기본 일본어 명령과 범용 콘셉트형 명령으로 제한한다.

## 기술 조건

- 브라우저가 speech recognition API를 지원해야 음성 시동이 동작한다.
- 마이크 권한이 필요하다.
- 미지원 또는 권한 거부 시 수동 시동으로 동일 흐름을 진행할 수 있어야 한다.

## 개인정보 조건

- 원시 오디오를 저장하지 않는다.
- transcript를 서버로 전송하지 않는다.
- 인식 결과는 홈 화면 상태 표시와 즉시 라우팅 판단에만 사용한다.

## 구현 중단 조건

- 전투 스킬 발동으로 연결하라는 요구가 들어오면 skill domain source가 승인될 때까지 중단한다.
- 계정별 음성 명령 저장이 필요하면 별도 개인정보 보존 정책이 먼저 필요하다.
- 브라우저 밖 STT provider가 필요해지면 비용, 키 관리, 보존 정책을 별도 결정한다.
