# v4 에픽

v4는 일본어 음성 시작 명령을 기존 홈 화면 진입 흐름에 얇게 연결합니다. 전투 중 스킬 사용이나 skill/resource domain 구현은 아직 승인된 소스가 없으므로 포함하지 않습니다.

## V4-E1: Japanese Voice Startup Command

- 목표: 사용자가 일본어 음성 명령으로 앱의 시작 흐름을 진행할 수 있게 한다.
- 범위: 마이크 시작 버튼, `ja-JP` 인식, 명령어 매칭, 성공/실패 상태, 마이크 없는 시작 경로.
- 제외: 전투 액션 제출, 스킬 발동, 억양/발음 평가.
- 수용 신호:
  - 지원 브라우저에서 마이크 버튼을 눌러 일본어 transcript를 받을 수 있다.
  - 등록 명령어 포함 시 현재 계정 상태에 맞는 다음 행동으로 이동한다.
  - 미지원/권한 거부/실패 시 마이크 없이 시작할 수 있다.

## V4-E2: Command Customization Readiness

- 목표: 추후 사용자별 명령어 등록으로 확장 가능한 구조를 남긴다.
- 범위: 명령어 배열과 매칭 함수를 UI에서 분리.
- 제외: 저장형 커스텀 명령어, 계정별 명령어 API.
- 수용 신호:
  - 명령어 추가/교체가 UI 컴포넌트 수정 없이 가능하다.
  - 매칭 함수 단위 테스트가 있다.

## V4-E3: Voice Startup Release Evidence

- 목표: 무료 내장 STT, fallback, 개인정보 경계를 문서화한다.
- 범위: 기술스택, 스토리, 구현 순서, 선행조건, 구현 기록.
- 수용 신호:
  - 한국어/영어 문서가 쌍으로 존재한다.
  - 특정 제공자, 작품 고유 표현, 원시 음성 저장이 기본 설계에서 제외된다.
