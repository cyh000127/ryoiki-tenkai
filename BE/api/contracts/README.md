# API Contracts

This directory is the canonical wire-contract source for the backend API, async event schemas,
and shared MVP catalog fixtures.

Generated frontend artifacts belong under `FE/app/src/generated/api-client`.
Do not redefine transport payloads in feature or page code.

`contracts/catalog/mvp-gesture-catalog.json` is the shared fixture for the normalized MVP
gesture token set and the default skill-to-sequence mapping.
